# 版本历史

## v3.0 (当前版本)
- 发布时间：2026/05/19
- **全新训练面板**：
  - 独立训练页面，支持YOLO11模型训练全流程
  - 数据集上传（zip压缩包）与管理
  - 预训练模型选择与刷新（`pre_models` 目录）
  - 训练环境检测：显示ultralytics版本、torch版本、CUDA支持状态
  - 实时进度展示（轮数、进度、mAP@0.5、已用时间）与日志滚动
  - 自动恢复训练（断点续训），基于last.pt恢复
  - 模型测试：上传图片验证效果，结果支持点击放大
  - 模型切换时显示指标参数（mAP、Precision、Recall）与训练参数
  - 模型下载
- **训练状态持久化**：
  - 页面刷新后自动恢复训练状态、按钮状态和进度
  - SocketIO重连后自动同步训练进度
- **数据集训练修复**：
  - 训练前自动修正data.yaml路径、标签class_id越界、清理缓存文件
- **文件存储优化**：
  - 预训练模型目录迁移到根目录 `pre_models`
  - 标注图片统一存储到 `uploads/samples`
  - 训练临时文件存放于 `tmp` 目录
- **离线部署**：Socket.IO本地化，所有页面添加favicon
- **UI优化**：删除安装/卸载按钮，进度区域压缩，图标按钮精简
